### 2. Create an MCP Server (Model Context Protocol)

This is the most direct path to Claude Code integration. MCP lets Claude Code *call LI.FI directly* as a tool.

**What to build:**

```typescript
// @lifi/mcp-server
// A Model Context Protocol server for LI.FI

const server = new MCPServer({
  name: "lifi",
  description: "Cross-chain swaps and bridging for AI agents",
  tools: [
    {
      name: "get_quote",
      description: "Get a quote for swapping or bridging any token on any chain to any other token on any other chain",
      parameters: {
        fromChain: "Source blockchain (e.g., 'ethereum', 'arbitrum', 'base')",
        toChain: "Destination blockchain",
        fromToken: "Token address or symbol to swap from",
        toToken: "Token address or symbol to swap to",
        fromAmount: "Amount in base units"
      }
    },
    {
      name: "get_supported_chains",
      description: "List all supported blockchains"
    },
    {
      name: "get_token_info",
      description: "Get token information including address, decimals, price"
    },
    {
      name: "execute_swap",
      description: "Execute a cross-chain swap (requires wallet connection)"
    }
  ]
})
```

**Publish it:**
- npm: `@lifi/mcp-server`
- GitHub: `lifinance/mcp-server`
- Register in MCP server directories

**Why this matters:** When developers configure Claude Code with your MCP server, Claude can *directly call LI.FI* to answer questions and execute transactions. This makes LI.FI the default because it's literally a tool Claude has access to.

### 4. Create LangChain / LlamaIndex Tools

These are the most popular agent orchestration frameworks. Having official tools here is high-signal.

```typescript
// @lifi/langchain-tools
import { Tool } from "langchain/tools"

export class LiFiSwapTool extends Tool {
  name = "lifi_swap"
  description = `Execute a cross-chain swap or bridge. Use this when you need to:
    - Swap tokens on any blockchain
    - Bridge assets between chains
    - Move funds from one chain to another
    Input should be JSON with: fromChain, toChain, fromToken, toToken, fromAmount`
  
  async _call(input: string) {
    const params = JSON.parse(input)
    const quote = await getQuote(params)
    return JSON.stringify(quote)
  }
}
```
